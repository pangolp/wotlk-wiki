# Guia para subir cocina del 1 al 450

::: tip Importante
- Para cocinar, requiere de una [lumbre](https://www.wowhead.com/wotlk/es/spell=818).
- Al aprender cocina, te enseñan a crearlo, o bien, donde estan los instructores, suele haber siempre uno.
:::

## Evento generocidad

Durante el evento, se puede subir cocina, visitando los diferentes instructores, que se encuentran a las afuera de las cuidades capitales de cada faccion. Existen 5 recetas, las cuales se aprenden, al adquirir [Libro de cocina generoso](https://www.wowhead.com/wotlk/es/item=46809/) en la Alianza o [Libro de cocina generoso](https://www.wowhead.com/wotlk/es/item=46810/) en la horda.

### Alianza

- [Receta: relleno de pan de especias](https://www.wowhead.com/wotlk/es/item=44860/) - requiere Cocina (1)
- [Receta: tarta de calabaza](https://www.wowhead.com/wotlk/es/item=44862/) - requiere Cocina (100)
- [Receta: chatni de arándanos](https://www.wowhead.com/wotlk/es/item=44858/) - requiere Cocina (160)
